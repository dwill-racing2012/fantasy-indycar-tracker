# My First Project: [Fantasy Indycar Tracker]

## 📌 Project Overview
"A tool to track IndyCar driver statistics and fantasy points using Excel data."

## 🚀 Features
- [ ] Feature 1: Automated data calculation
- [ ] Feature 2: Visual dashboard for trends
- [ ] Feature 3: Exportable summary reports

## 🛠️ Technologies Used
- **Git/GitHub**: Version control and hosting
- **Excel/Google Sheets**: Data processing
- **Markdown**: Documentation

## 📚 Study Notes / Resources
- [Indycar Fantasy](https://www.indyfantasy.com)
- [Indycar Website](https://www.indycar.com)

## ✍️ How to Use
1. Clone this repository to your computer.
2. Open the `main-file.xlsx`.
3. Update the data in the 'Input' tab.
